📖 Cara Install APK
1
Tap tombol Download APK di atas — file akan tersimpan di folder Downloads.
2
Buka file manager → Downloads → tap MNN-Learning-latest.apk.
3
Jika muncul peringatan "sumber tidak dikenal", pilih Izinkan dari sumber ini untuk browser kamu.
4
Tap Install → selesai! Ikon MNN Learning akan muncul di launcher.
5
Update: cukup download APK terbaru dan install di atas versi lama — data tidak akan terhapus.
